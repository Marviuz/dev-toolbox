# Node Version Manager

[Node Version Manager (nvm)](https://github.com/nvm-sh/nvm) is a tool to manage multiple versions of Node.js on a single machine.

## Installation

```sh
scoop install nvm
```

## Usage & Commands

### `nvm install`

Install a version of Node.js

```sh
nvm install 10.15.3
```

_Note: You can use `nvm install lts` to install the latest LTS version of Node.js._

### `nvm use`

Use a version of Node.js

```sh
nvm use 10.15.3
```

_Note: You can use `nvm use lts` to use the latest LTS version of Node.js._

### `nvm list`

List installed versions of Node.js

```sh
nvm list
```

### `nvm uninstall`

Uninstall a version of Node.js

```sh
nvm uninstall 10.15.3
```
